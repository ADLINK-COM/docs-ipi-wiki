COM-HPC Mini Base Carrier Board

Industrial-grade reference carrier board for standard COM-HPC Mini modules


This page aims to highlight the capabilities of the COM-HPC Mini Base Carrier Board. For a more comprehensive information, please refer to its user manual.

Download User Manual


Specifications

Core Module Interface

  • PICMG COM-HPC Revision 1.2

  • Support COM-HPC Mini Type modules

Dimension

  • 305 mm x 244 mm (ATX)

Expansion Busses

  • 1x PCI Express x8 slot (PCIe lane 8-15) - Note: An add-on card is required for 2 PCIe x4 signal lanes.

  • 1x PCI Express x4 slot (PCIe lane 0-3)

  • 1x M.2 M-key connector (PCIe lane 4-7)

POST LEDs

  • Onboard IO expander for BIOS POST code data on I2C bus

Secondary BIOS

  • Onboard socket for secondary SPI flash

COM Express Connectors

  • Single 400-pin (10 mm height)

DP

  • 1x DP connector (DDI0)

USB4

  • 2x ports (Type C) on rear I/O*

Flat Panel

  • 40-pin connector eDP

Audio

  • Mic-in/Line-in/Line-out on rear I/O

SATA

  • 2x SATA3.0 (SATA0/SATA1)

USB

  • 2x USB 3.2 gen2 type A on rear I/O

  • 2x USB 2.0 type A on rear I/O

  • 2x USB 2.0 with 9-pin header for front panel

LAN

  • 2x ports RJ45 10GBase-T up to 10 GbE on Rear I/O

GIPO

  • GPIO 12-pin header GPIO

Serial Port

  • 2x 10-pin header (from module)

  • 2x 10-pin header (from Super I/O)

FAN

  • 1x 4-pin fan connector from module

  • 2x 4-pin fan connector from Super I/O

MIPI-CSI

  • 2x CSI connector (CSI0/CSI1)

Feature Connectors

  • Headers for SMBus, I2C, eSPI

  • Header for front panel usage

  • Jumpers for module control signals

Miscellaneous

  • Power button, reset button, sleep button, status LEDs (Lid button and Buzzer by project support)

*Interface and signal compatibility is subject to the module and carrier used.
