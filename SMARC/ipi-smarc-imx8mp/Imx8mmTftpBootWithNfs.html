TFTP Boot with NFS

Below describes how to boot a target device from TFTP with NFS.

Before you start

  • Prepare your Linux host device for the server setup.
  • Make sure the target device(I-Pi SMARC IMX8MM) and the host device are connected to the same network.

Note: The following procedures have been tested with Ubuntu 18.04 and 20.04 only.
