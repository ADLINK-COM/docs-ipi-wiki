How to Upgrade BIOS on I-Pi SAMRC Elkhart Lake

This document provides detailed instructions for upgrading the BIOS on I-Pi SMARC Elkhart Lake.

Prerequisites

  • Download the latest BIOS file click here.

  • Copy the downloaded BIOS file to a USB pen drive.

    Changing the Boot Option

  • Ensure your targeted device(I-Pi SMARC Elkhart Lake) is equipped with all necessary devices and cables such as a keyboard, mouse, USB pen drive, Ethernet cable, and HDMI.

  • While the device is powered on, you’ll see the Adlink logo as shown in below figure and you’ll need to press the ‘Del’ key on your keyboard to enter the boot menu.

  • In the ‘main section’ of the boot menu, you will see the current BIOS version (0.15.10) displayed as shown below.

  • You can move into the boot section by pressing the right arrow key.

  • As shown in the following figure, click Boot Option #1 and select your USB pen drive as the first option.

  • Next, move to the ‘save & exit’ section and click ‘Save Changes and Reset’ as shown below figure.

Entering into the Shell Mode

  • This time do not touch anything, it will automatically enter shell mode and type the following commands, as shown in the following figures.
  • Upon typing all the commands shown above, it will enter the next stage as shown below figure, press ‘Enter’ to start the BIOS upgrade process.
  • After the upgrade is complete, the screen will display some messages as shown below.
  • You can now reset the targeted device by pressing the reset button.
  • To access the boot menu, press the key ‘Del’ on the keyboard. A BIOS version will appear as (1.00.10).
  • In the end, your BIOS version was successfully upgraded. Later, the boot menu options can then be modified as desired.
