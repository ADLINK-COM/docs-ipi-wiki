OSM-EB4


Specifications

Core System

SoC

  • MediaTek Genio 510 dual-core Arm® Cortex-A78 & quad-core Arm® Cortex-A55

NPU

  • MediaTek 5th Gen NPU, up to 3.2 TOPS

Architecture

  • 2x Arm® Cortex-A78 + 4x Arm® Cortex-A55

Memory

  • 2GB LPDDR4

Storage

  • 32GB eMMC

Security

  • Arm® TrustZone®

Video

GPU Core

  • Arm Mali-G57 MC2

HDMI

  • 1x HDMI interface

MIPI DSI

  • 1x MIPI DSI via LVDS connector, 4 lanes

DP

  • 1x DP interface (Reserved Pin)

eDP

  • 1x eDP interface, 2 lanes

Camera

  • 1x 22-pin CSI connector, with 4 lanes available.

Audio

Codec

  • TI TLV320AIC3107

Ethernet

LAN

  • 2x GbE (1x TSN capable)

Connectors/Headers

Display

  • 1x HDMI
  • 1x DSI
  • 1x DP
  • 1x eDP

Audio

  • 1x 3.5mm Audio jack

Network

  • 2x RJ45

USB

  • 2x USB 2.0 Host
  • 2x USB 3.0 Host
  • 1x USB 2.0 OTG

SD Card Slot

  • 1x microSD

M.2 Card Slots

  • 1x M.2 M-key PCIe

UART

  • 3x UART on header

SPI

  • 2x SPI on header

I2C

  • 2x I2C on header

GPIO

  • 8x GPIO on header

PWM

  • 4x PWM on header

ADC

  • 2x ADC on header

Power

Input

  • 15 to 30 VDC (Adapter: 19VDC)

Connector

  • DC jack

Operating Systems

Standard Support

  • Yocto Linux BSP
  • Android (by project)
  • Ubuntu (by project)

Extended Support

  • Foundries.IO Linux microPlatform (LMP)

Mechanical and Environmental

Form Factor

  • 4” EPIC

Dimensions

  • 165mm x 115mm

Operating Temperature

  • Standard: 0°C to +60°C

  • Rugged: -40°C to +85°C (optional)

Humidity

  • 5-90% RH operating, non-condensing

  • 5-95% RH storage (and operating with conformal coating)

Shock and Vibration

  • IEC 60068-2-64 and IEC-60068-2-27, MIL-STD-202 F, Method 213B, Table 213-I,
    Condition A and Method 214A, Table 214-I, Condition D
