Serial Over LAN (SOL) Commands:

Console redirection: Commands to start or stop SOL sessions.

Command Description
Starting SOL Session
To start a SOL session using IPMI, you generally use the sol activate command.		 ipmitool -H <BMC_IP> -U <username> -P <password> sol activate
- <BMC_IP>: Replace this with the IP address or hostname of your BMC.
- <username> and <password>: Replace these with your BMC login credentials.		 Example:
ipmitool -H 192.168.1.100 -U admin -P password sol activate
This command initiates a SOL session with the BMC.
Stopping SOL Session
To stop a SOL session, you typically use the sol deactivate command.		 ipmitool -H <BMC_IP> -U <username> -P <password> sol deactivate Example:
ipmitool -H 192.168.1.100 -U admin -P password sol deactivate
This command terminates the current SOL session with the BMC.

Additional Notes

  • IPMI Tool (ipmitool): Ensure that ipmitool is installed on your system. It’s a common command-line interface for managing BMCs and provides various capabilities including SOL session management.

  • Authentication: Replace <username> and <password> with your actual BMC login credentials.

  • BMC IP Address: Use the correct IP address or hostname of your BMC.

These commands should help you start and stop SOL sessions via the BMC using IPMI commands as outlined in the user guide provided by BMC for your specific hardware or software setup. Always refer to the latest BMC documentation or user guide for any updates or specific instructions tailored to your system.


Keyboard and video redirection: Commands to redirect keyboard and video input/output.

These commands can vary slightly depending on the specific BMC implementation, but generally, they include functionality for managing remote keyboard and video interfaces. Here’s a list of common commands and their explanations:


Keyboard Redirection Commands

Command Description
Start Keyboard Redirection start_kbd_redirection Initiates the redirection of keyboard input from the local console to the remote server.
Stop Keyboard Redirection stop_kbd_redirection Terminates the ongoing keyboard redirection session, returning control to the local console.
Send Key Stroke send_key_stroke <key> Sends a specific key stroke to the remote server. <key> is the key you want to send, such as Ctrl, Alt, Del, etc.
Send Key Combination send_key_combination <key1>+<key2> Sends a combination of keys to the remote server, such as Ctrl+Alt+Del.


Video Redirection Commands

Command Description
Start Video Redirection start_video_redirection Initiates the redirection of video output from the remote server to the local console.
Stop Video Redirection stop_video_redirection Ends the ongoing video redirection session.
Capture Screenshot capture_screenshot Captures a screenshot of the current video output from the remote server and saves it to a specified location.
Adjust Video Quality adjust_video_quality <quality> Adjusts the quality of the video redirection stream. <quality> can be values like low, medium, or high.
Full-Screen Mode toggle_full_screen Toggles between full-screen mode and windowed mode for video redirection.


General Redirection Commands

Command Description
Start All Redirection start_all_redirection Simultaneously starts both keyboard and video redirection sessions.
Stop All Redirection stop_all_redirection Simultaneously stops both keyboard and video redirection sessions.
Set Redirection Timeout set_redirection_timeout <time> Sets a timeout value for the redirection session. <time> is specified in seconds or minutes.

These commands are typically executed through a BMC interface, such as a web console or command-line utility. Ensure you refer to the specific BMC documentation for your hardware, as the exact syntax and available commands may vary.


Terminal modes: Commands to adjust terminal modes and settings.

Terminal Modes Commands

Command Description
Set Terminal Mode set_terminal_mode <mode> Sets the terminal to a specific mode. <mode> can be values like text, graphics, or secure.
Enable Echo enable_echo Enables echoing of characters typed by the user, which is useful for normal interaction.
Disable Echo disable_echo Disables echoing of characters typed by the user, often used for password entry or secure input.
Set Baud Rate set_baud_rate <rate> Sets the baud rate for serial communication. <rate> is the desired baud rate, such as 9600, 19200, etc.
Set Parity set_parity <parity> Configures the parity bit for serial communication. <parity> can be none, even, odd, mark, or space.
Set Data Bits set_data_bits <bits> Sets the number of data bits per character for serial communication. <bits> can be values like 7 or 8.
Set Stop Bits set_stop_bits <bits> Configures the number of stop bits used in serial communication. <bits> can be 1 or 2.
Set Flow Control set_flow_control <type> Configures flow control for serial communication. <type> can be none, hardware, or software.
Set Terminal Size set_terminal_size <rows> <cols> Sets the size of the terminal window. <rows> and <cols> specify the number of rows and columns, respectively.
Reset Terminal reset_terminal Resets the terminal settings to their default values.
Set Terminal Encoding set_terminal_encoding <encoding> Sets the character encoding for the terminal. <encoding> can be values like UTF-8, ISO-8859-1, etc.
Enable/Disable Line Wrapping toggle_line_wrapping Toggles the line wrapping setting of the terminal.


Terminal Settings Commands

Command Description
Set Terminal Color Scheme set_color_scheme <scheme> Changes the terminal’s color scheme. <scheme> can be values like default, light, dark, etc.
Set Cursor Style set_cursor_style <style> Sets the cursor style in the terminal. <style> can be block, underline, or beam.
Enable/Disable Bell toggle_bell Toggles the audible bell on or off.
Set Terminal Title set_terminal_title <title> Sets the title of the terminal window. <title> is the desired title string.
Enable/Disable Scrollback Buffer toggle_scrollback_buffer Toggles the scrollback buffer on or off, allowing you to review previous output.
Set Scrollback Buffer Size set_scrollback_buffer_size <size> Sets the size of the scrollback buffer. <size> specifies the number of lines to keep in the buffer.
Set Terminal Font Size set_font_size <size> Changes the font size of the terminal text. <size> is the desired font size.
Enable/Disable Blinking Cursor toggle_blinking_cursor Toggles the blinking cursor feature on or off.

These commands help in configuring and customizing the terminal to suit specific needs and preferences. The exact syntax and availability of these commands might vary depending on the BMC implementation, so it’s advisable to refer to the specific documentation for your hardware.
