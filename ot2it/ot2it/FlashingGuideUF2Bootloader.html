Bootloader Flashing Guide

This document provides details of the UF2 bootloader for Adlink OT2IT.

[!NOTE]

If the bootloader already exists inside MCU, please skip the below step.

  • Download the OT2IT Bootloader on your host machine (suggest that all of the operations are on a Windows environment)

Hardware and Setup

  • Connect the J-Link cable (we’re using J-LINK EDU Mini) on the board and your host machine

  • Connect the power cable to power on the board as shown below

Firmware Download

  • Download and install the latest version of Microchip Studio [Offline installer] Microchip Studio for AVR® and SAM Devices | Microchip Technology on your host machine

[!NOTE]

First-time installers should follow the below instructions.

  • Accept the license agreement

  • Verify that architectures are enabled by default and press the ‘Next’ button

  • Verify that the extensions have been chosen and click ‘Next’

  • To continue, click ‘Refresh’ and ‘Next’ after verifying all system validations

  • Click ‘Next’ for all installation setups as shown in the example below

  • Install all the necessary device software as shown in the example below

  • Once successfully installed, open Microchip Studio application and click ‘Device Programming’ under ‘Tools’

Flashing Configurations

  1. Select the ‘J-Link’ tool and ‘Device’
  2. Select ‘Interface’ and click ‘Apply’
  3. Read the device signature by clicking ‘Read’
  4. Select .bin file
  5. Click the ‘Program’ button
  6. The messages are a sign of successful flashing
  • Once flashed successfully, disconnect the J-LINK cable and power cable.
