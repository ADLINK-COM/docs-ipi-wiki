Digital Signals

Connector: DI- CH1-3
Ref: CN2302		 MCU PIN signal description
Pin Signal Type Pin
1 24V P
2 DI4 DI MCU, PC14
3 24V P
4 DI5 DI IO Exoabder, I.O1 GPIO | I/01
5 24V P
6 DI6 DI6 IO Expander, I/O2 GPIO | I/02
7 GND P Ground signal
Connector: DI- CH4-5
Ref: CN2301		 MCU PIN signal description
Pin Signal Type Pin
1 24V P
2 DI0 DI PA06
3 24V P
4 DI1 DI PA07
5 GND P Ground signal
Connector: DI- CH6-7
Ref: CN2306		 MCU PIN signal description
Pin Signal Type Pin
1 24V P
2 DI2 DI PC07
3 24V P
4 DI3 DI PA10
5 GND P Ground signal
Connector: DO- CH1-2
Ref: CN2303		 MCU PIN signal description
Pin Signal Type Pin
1 24V/12V/5V P Input voltage, selectable via CN2901
2 DO0 DO PC10
3 GND P Ground signal
4 14V/12V/5V P Input voltage, selectable via CN2901
5 DO1 DO PD11
6 GND P Ground signal
Connector: DO- CH3-4
Ref: CN2308		 MCU PIN signal description
Pin Signal Type Pin
1 24V/12V/5V P Input voltage, selectable via CN2901
2 DO2 DO PB14
3 GND P Ground signal
4 14V/12V/5V P Input voltage, selectable via CN2901
5 DO3 DO PC18
6 GND P Ground signal
Connector: DO- CH5-6
Ref: CN2309		 IO Expander (SX1509B)
 signal description
Pin Signal Type Pin
