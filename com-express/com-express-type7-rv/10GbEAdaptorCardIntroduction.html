10GbE Adaptor Card Introduction

Introduction

COM Express Type7 Ryzen V3000 devkit comes along with a dedicated 10Gbe Ethernet card (10G SFP+ or 10GBASE-T) that allows users to easily emulate the functionality of their end product for software development and hardware verification. This page aims to highlight the capability of the 10G Ethernet cards that are dedicated for COM Express Type7 Ryzen V3000 devkit.

  • 10G SFP+ (CEI-2x10G-SFP+)
  • 10GBASE-T (CEI-2x10GBASE-T)

10G SFP+ (CEI-2x10G-SFP+)

The 10G SFP+ Ethernet Card (CEI-2x10G-SFP+) in the COM Express Type7 Ryzen V3000 devkit is a 10GbE SFP+ network adapter card that follows PICMG COM.0 R3.1 specification and can deliver fast 10Gbps network access. It converts up to two 10GBASE-KR interfaces & related CEI style sideband signals to two 10G SFP+ interfaces at SFP+ connectors, through Intel CS4227 PHY and multiple IO expanders.

Features

  • Support up to 2x 10GbE SFP+ interfaces
  • COM.0 R3.1 Type 7 compliant
  • Dedicated for Express-VR7 Module

Specifications

Interface

Input

  • Up to two 10GBASE-KR & related CEI style sideband signals from a dedicated COM Express Type 7 through a 164-pin edge connector (PCIe x16 slot PCIEKR on Express-BASE7 Plus)

Output

  • Up to two 10G SFP+ interfaces at two SFP+ connectors

10GbE PHY

PHY

  • Chip: Intel CS4224
  • Converts 10GBASE-KR and sidebands to 10G SFP+ interface

PHY FW on SPI ROM

  • Dedicated ROM for PHY firmware store

10GBASE-T(CEI-2x10GBASE-T)

The 10GBASE-T Ethernet Card (CEI-2x10GBASE-T) in the COM Express Type7 Ryzen V3000 dev kit is a 10GbE BASE-T network adapter card that follows PICMG COM.0 R3.1 specification and can deliver fast 10Gbps network access. It converts up to two 10GBASE-KR interfaces, and related CEI sideband signals, to two 10GBASE-T interfaces via RJ45 connectors, through Marvell AQR113C PHY and multiple I/O expanders.

Features

  • Support up to two10GbE copper interfaces (10GBASE-T)
  • COM.0 R3.1 Type 7 compliant
  • Dedicated for Express-VR7 Module

Specifications

Interface

Input

  • Up to two 10GBASE-KR & related CEI style sideband signals from a dedicated COM Express Type 7 through a 164-pin edge connector (PCIe x16 slot PCIEKR on Express-BASE7 Plus)

Output

  • Up to two 10GBASE-T interfaces via two RJ45 connectors

10GbE PHY

PHY

  • Chip: Marvell AQR113C
  • Converts 10GBASE-KR and sidebands to 10GBASE-T

PHY FW on SPI ROM

  • Dedicated ROM for PHY firmware store

Installation

The following picture shows where to install the 10GbE Adapter card on the Express-BASE7 Plus carrier board. User could simply insert the card into the dedicated slot (PCIe KR) of the Express-BASE7 Plus carrier board.

Mechanical and Environmental

Form Factor

  • PICMG COM Express Rev 3.1 (COM.0 R3.1), Type 7

Dimensions

  • 168 mm x 107 mm (PCB only, including edge connector)

Operating Temperature

  • Standard: 0°C to 85°C
