Express-BASE 7 Plus Carrier Board

Industrial Prototyping / Reference Carrier for Off-the-shelf COM Express Type 7 Module

This page aims to highlight the capability of the Express-BASE 7 Plus Carrier Board. For the comprehensive information please refer to the user manual.

Download User Manual

Introduction

The Express-BASE7 Plus carrier board is compatible with Basic and Compact form factor of COM Express Type 7 modules pin-out , featuring one PCI Express x16 slot with proprietary pinout for inserting a 10GbE Ethernet Adapter Card (supports up to four 10G signals, dependent on COM Express module specifications), one PCI Express x16 Gen4 slot, and two PCI Express x8 slots.

The Express-BASE7 Plus is based on the PICMG COM Express (COM.0 Rev. 3.1) specification.

Features

  • Reference carrier board based on COM Express Rev 3.1

  • Support R3.1 Type 7 modules Basic / Compact size

  • Support 10GbE adapter card (up to 4x 10GBASE-KR and sideband) via PCIe x16 slot

  • Additional Ethernet port on carrier, up to 2.5GbE

  • One PCI Express x16 Gen4 slot with 2nd PCIe clock

  • Additional one PCI Express x4 slot, three PCI Express x1 slots and SATA/USB ports

Note: * PCIe speed support depends on the module


Specifications

Expansion Bus

  • 32x PCIe Lanes*
    • 1x PCIe x16 slot for 10GbE Ethernet Adapter Card
    • 2x PCIe x8 slots

LAN

  • Up to 4x 10GBASE-KR and related sideband signals 1 PCI Express x16 slot routes signals from COM Express module

  • 10/100/1000BASE-T compatible RJ-45 on I/O panel

USB

  • 4x USB 3.0/2.0 ports on I/O

SATA

  • 2x SATA ports on rear

Serial Port

  • 2x DB-9 on I/O panel (from Winbond Super I/O)
  • 2x onboard 10-pin header (from COM Express module)

KB/Mouse

  • 2x 6-pin mini-DIN

Feature Connectors

  • SMBus, I2C, module control signals

NC-SI and IPMI BMC

  • Equip IPMI MNC(MiniMBC) for remote management

Primary LPC Based Super I/O

  • Use Winbond W83627DHG-PT on LPC bus.

Secondary BIOS

  • Onboard socket for secondary SPI flash. Selection of active BIOS can be made by jumper settings.

Integrated Debug Port

  • Onboard diagnostics for BIOS POST code data and address on LPC bus

Miscellaneous

  • Reset, Power LED, HDD LED, Buzzer

Power

  • 100-240V
* The Interface and signal compatibility is subject to the module and carrier used.

Mechanical and Environmental

  • Form Factor: PICMG® COM.0 Rev. 3.1

  • Dimensions: 305 mm x 244 mm

  • Operating Temperature:

    • Standard: 0°C to 60°C
    • Rugged: -40°C to 85°C (optional)

  • Humidity:

    • 5-90% RH operating, non-condensing

    • 5-95% RH storage (and operating with conformal coating)

  • Shock and Vibration:

    • IEC 60068-2-64 and IEC-60068-2-27, MIL-STD-202 F,
    • Method 213B, Table 213-I, Condition A and Method 214A,
    • Table 214-I, Condition D

  • HALT

    • Thermal Stress, Vibration Stress, Thermal Shock and Combined Test
